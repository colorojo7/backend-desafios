E-commerce Mauro Leonardi

Rutas:
Productos:
  • GET localhost:8080/api/products => Ver todos los productos disponibles
  • POST localhost:8080/api/products/ => Agregar un nuevo producto al e-commerce (debe completar todos los datos)
  • GET localhost:8080/api/products/id => Ver el producto solicitado por parámetro
  • PUT localhost:8080/api/products/id => Modificar uno o más datos del producto indicado por parámetro
  • DELETE localhost:8080/api/products/id => Eliminar el producto indicado por parámetro
Carritos:
  • GET localhost:8080/api/carts => Ver todos los carritos creados
  • POST localhost:8080/api/carts => Crear un nuevo carrito (se podrá darle un "name" al carro)
  • GET localhost:8080/api/carts/id => Ver el carrito solicitado por parámetro
  • POST localhost:8080/api/carts/:cid/products/:pid => Permitirá agregar al carrito indicado el producto seleccionado
